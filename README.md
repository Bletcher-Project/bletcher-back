## Technology

- Node.js
- Express
- Sequelize
- Typescript

## Run

실행 전, 루트 경로에 .env.example 파일을 참고한 .env 파일을 생성해주세요.

Development Mode

```bash
yarn install
yarn dev
```

Production Mode

```bash
yarn install
yarn build
yarn start
```
